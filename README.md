[![Ruby](https://img.shields.io/badge/Ruby-2.7.0-red)](#)

EuGame is a gem that supports set of CLI/TUI based 2d games.

## Games
- PvP TicTacToe <i>(last updated: 2020-05-31)</i>

## Install
```
gem install eugame
```

## Usage
```rb
require 'eugame'

game = EUGAME::EuGame.new
game.start
```
